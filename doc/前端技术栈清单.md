

## 核心框架

- `Vue 3`
  - 工作区所有前端/桌面端项目的统一基础框架，用于构建页面、组件和交互逻辑。
- `Electron`
  - 桌面应用运行容器，将前端页面封装为跨平台桌面应用并接入系统原生能力。用于 `aibot_clien_V3`、`aibot_client_web`、`fast_operations_frontend`。

---

## 路由

- `Vue Router`
  - 页面路由管理，负责路由跳转、守卫和模块拆分。绝大多数项目使用。
- `Vike` + `vike-vue`
  - Vue 生态的 SSR/同构页面框架，支持页面级路由组织和服务端渲染。用于 `cloud_control_user`。

---

## 状态管理

- `Pinia`
  - 全局状态管理，管理跨组件共享数据。所有项目使用。
- `pinia-plugin-persistedstate`
  - Pinia 持久化插件，将指定状态落到本地缓存（localStorage/sessionStorage）。所有项目使用。

---

## 网络请求与通信

- `Axios`
  - HTTP 请求库，负责接口通信。所有项目使用。
- `socket.io-client`
  - 实时通信客户端，基于 WebSocket 建立长连接。用于 `fast_operations_frontend`。

---

## UI 组件库

- `Element Plus`
  - Vue 3 生态通用 UI 组件库，负责表单、弹窗、按钮、布局等基础界面能力。所有项目使用。
- `@element-plus/icons-vue`
  - Element Plus 配套图标库。
- `vxe-table`
  - 企业级表格，适合复杂列表、可编辑表格、固定列、虚拟滚动等场景。多数项目使用。
- `vxe-pc-ui`
  - 与 `vxe-table` 配套的 PC 端组件能力补充。用于 `aibot-chat-web-v3`、`cloud_control_user`。
- `xe-utils`
  - `vxe-table` 配套工具库，提供数据处理辅助方法。用于 `fast_operations_frontend`。
- `@vxe-ui/plugin-export-xlsx`
  - 为 vxe-table 提供导出 xlsx 能力。用于 `aibot-chat-web-v3`、`cloud_control_user`。
- `vue3-tabs-chrome`
  - Chrome 风格标签页组件，适合多会话、多标签工作台。用于 `aibot_clien_V3`。
- `@imengyu/vue3-context-menu`
  - 右键菜单组件。用于 `aibot_clien_V3`、`aibot_client_web`。
- `virtual-list-vue3`
  - 虚拟列表组件，适合长消息流、大列表渲染优化。用于 `aibot_clien_V3`。
- `emoji-mart` + `@emoji-mart/data`
  - 表情选择器，支持 emoji 渲染与选择能力。用于 `aibot_clien_V3`。
