## 各目录概括

| 目录 | 说明 |
|------|------|
| `doc/` | 项目文档，包含需求规格、技术规范、已知问题及开发规范 |
| `frontend/` | Vue 3 + Vite 前端，LogicFlow 画布编辑器，通过 File System Access API 直接读写本地工作流文件，Vite 开发服务器运行在 `:5173` 并代理 `/api` 到后端 |
| `frontend/src/components/` | 页面级 Vue 组件，核心为 `WorkflowEditor`（画布）和 `WorkspaceManager`（工作区管理） |
| `frontend/src/services/` | 单例模式服务层，封装文件系统、协作、WebSocket、API 调用等能力 |
| `src/` | NestJS 11 + Fastify 后端，运行在 `:9000`，提供 REST API、WebSocket 协作网关和 MCP AI 集成 |
| `src/project/` | 项目元数据 CRUD，持久化到 PostgreSQL |
| `src/node/` | 工作流节点管理，支持批量状态更新、执行历史记录、全量同步 |
| `src/collaboration/` | 基于 WebSocket 的多人实时协作，含房间、用户、心跳、操作日志管理 |
| `src/mcp/` | MCP 协议服务，向 AI IDE（如 Cursor/Windsurf）暴露工具和资源，支持订阅推送 |
| `src/services/` | 通用基础服务，含定时任务调度（`task-scheduler`）、文件操作、Schema 校验等 |
| `src/config/` | YAML 配置加载，按 `NODE_ENV` 区分环境，Zod 强类型校验 |
| `src/migrations/` | MikroORM 数据库迁移脚本，管理 schema 版本 |
